Expense Tracker:
A simple and beginner-friendly Python project that helps users record and manage their daily expenses. The program allows you to add expenses, categorize them, view totals, and understand your spending patterns. It is useful for anyone who wants basic financial tracking and is a great project for practicing Python fundamentals

Features:
Add daily expenses
Categorize spending
View total and category-wise summaries
Simple and easy-to-use interface

Technologies Used:

Python
File handling / Lists / Functions
How to Run:
1. Clone the repository
2. Run the Python file:
 python expense_tracker.py
3. Follow the on-screen menu


Future Improvements:
Add a graphical interface
Export data to CSV
Monthly reports and charts
